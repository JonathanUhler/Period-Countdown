# CREATING THE INSTANCE

Buy a domain and create a hosted zone

Launch an EC2 instance

Allocate an Elastic IP to the EC2 instance

Create an A record pointing at the Elastic IP

Create a inbound rule under Security Groups for HTTP and HTTPS



# ACCESSING THE SERVER

ssh -i <key> ubuntu@<DNS-server>
